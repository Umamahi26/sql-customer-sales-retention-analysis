# sql-customer-sales-retention-analysis
End-to-end SQL data analysis project focusing on customer behavior, revenue trends, retention insights, and KPI reporting using joins, aggregations, CTEs, and window functions.
# Project Title
Customer Sales & Retention Analysis using SQL
# Project Overview
This project analyzes e-commerce sales data to generate insights on customer purchasing behavior, revenue trends, and retention patterns using SQL analytical techniques.
The objective is to simulate real-world business scenarios and demonstrate practical SQL skills required for Data Analyst roles.
# Business Problem
Organizations need to understand:

  - Which customers generate maximum revenue
  
  - Monthly sales performance trends
  
  - Customer engagement and inactivity patterns
  
  - Repeat vs one-time customer behavior
  
This analysis helps support data-driven marketing and retention strategies.
# Dataset Description
The project uses two relational tables:

# Customers Table:

  * customer_id
  
  * customer_name
  
  * city
  
  * signup_date

# Orders Table:

  *  order_id
  
   * customer_id
  
  *  order_date
  
  *  amount

The dataset represents simulated e-commerce transactions.

#  SQL Skills Demonstrated
* Data filtering and sorting

* Aggregation and grouping

* Table joins

* Subqueries and CTEs

* Window functions (ROW_NUMBER, RANK, LAG)

* KPI calculations and customer segmentation

# Key Analysis Performed
* Total and monthly revenue trend analysis

* Identification of top revenue-generating customers

* Revenue contribution percentage per transaction

* Latest order activity tracking

* Customer inactivity detection

* Repeat vs one-time customer classification

# Business Insights
* Hyderabad customers contributed highest cumulative revenue

* Top 3 customers generated ~60% of total sales

* Purchase frequency analysis shows declining engagement after Feb

* Average customer purchase gap is 25 days

* Premium orders (>₹7000) account for major revenue spikes

This project uses SQL to analyze sales performance and customer behavior, identifying key revenue drivers, monthly trends, and retention patterns. The insights support data-driven decisions in marketing, customer engagement, and regional strategy, demonstrating practical SQL skills for real-world business analysis.
